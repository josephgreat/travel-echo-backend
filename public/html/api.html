Creates new passport data

Content-Type: application/json Authorization: Bearer <token>
json 
{ "user": "string"; "passportNumber": "string"; "passportType": "PassportType"; "fullName": "string"; "nationality": "string"; "issueDate": "Date"; "expiryDate": "Date"; "placeOfIssue?": "string | null | undefined"; }
json 
{
  "success": true,
  "passport": {
    ...
  }
}

PUT

/users/me/passport

Updates the user's passport data

Content-Type: application/json Authorization: Bearer <token>
json 
{
  "user": "string",
  "passportNumber": "string",
  "passportType": "PassportType",
  "fullName": "string",
  "nationality": "string",
  "issueDate": "Date",
  "expiryDate": "Date",
  "placeOfIssue?": "string | null | undefined"
}
json 
{
  "success": true,
  "passport": {
    ...
  }
}

DELETE

/users/me/passport

Deletes the user's passport information

Authorization: Bearer <token>
json 
{
  "success": true,
  "message": "Passport data deleted successfully"
}

User: Profile

GET

/users/me/profile

Get user profile. This endpoint will also create a minimal profile if the profile doesn't already exist

Authorization: Bearer <token>
json 
{
  "success": true,
  "profile": {
    ...
  }
}

PUT

/users/me/profile

Update user profile. If the profile doesn't exist, it will be created with the provided data

Content-Type: application/json Authorization: Bearer <token>

  "gender": "MALE | FEMALE | OTHER",
  "location": "Abuja, Nigeria",

    "travelling",

  ],

    "English",

  ]
}
json 
{

    ...
  }
