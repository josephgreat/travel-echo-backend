Memories

GET

/memories

Gets the memories of the user with 5 memory images each

Name Type Required Description
search query false Searches for memories by title, location, or tag e.g search=<value>
title query false Specific search by title e.g title=<value>
location query false Specific search by location e.g location=<value>
tag query false Specific search by tag e.g tag=<value>
sort query false To sort the data e.g sort=createdAt,ASC
limit query false To limit the number of results e.g limit=10
skip query false To skip a number of results e.g skip=10
Authorization: Bearer <token>
